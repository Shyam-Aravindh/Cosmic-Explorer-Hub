# 🚀 Cosmic Explorer Hub

**Cosmic Explorer Hub** is a Python web dashboard built with **Streamlit** that connects to NASA's public APIs to deliver real-time space data and stunning astronomical media in a clean, interactive interface.

Developed as a submission for the **Stardance Challenge**.

---

## ✨ Features

### ☄️ Live Near-Earth Asteroid Tracking

Monitor asteroids making close approaches to Earth using NASA's **Near Earth Object Web Service (NeoWs)**. View information such as:

* Relative velocity
* Miss distance
* Estimated size
* Potentially hazardous status

### 🌌 Astronomy Picture of the Day (APOD)

Explore NASA's daily featured astronomy image along with its official explanation written by professional astronomers.

### 🔭 NASA Image & Video Archive

Search NASA's extensive media archive for historical images and videos from space missions, celestial objects, and scientific discoveries.

---

## 🛠️ Tech Stack

* **Python 3**
* **Streamlit** – Interactive web dashboard
* **Requests** – REST API integration
* **NASA Open APIs**

  * Near Earth Object Web Service (NeoWs)
  * Astronomy Picture of the Day (APOD)
  * NASA Image and Video Library

## 📦 Installation

### 1. Clone the repository

```bash
git clone https://github.com/Shyam-Aravindh/Cosmic-Explorer-Hub.git
cd Cosmic-Explorer-Hub
```

### 2. Install dependencies

```bash
pip install -r requirements.txt
```

### 3. Launch the application

```bash
streamlit run app.py
```
